## 💡 Рекомендации по развертыванию

### Для локальной разработки

1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Создать `.env` на основе `.env.example`
4. Добавить API ключ в Supabase
5. Запустить: `npm start`

### Для production

1. Убедиться, что API ключ добавлен в Supabase Secrets
2. Настроить RLS политики в Supabase
3. Проверить CORS настройки Edge Functions
4. Деплоить через Figma Make или собственный хостинг
